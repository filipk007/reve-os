# Reporting Cadence — {{client_name}}

## Weekly Report
- **Delivered:** Every Monday by 10am ET
- **Format:** Email summary + dashboard link
- **Covers:** Prior week activity, key metrics, pipeline movement

## Monthly Review
- **Delivered:** First week of each month
- **Format:** Slide deck + live walkthrough (30 min)
- **Covers:** Monthly KPIs, campaign performance, strategic recommendations

## KPIs Tracked
- Emails sent / opened / replied
- Meetings booked
- Pipeline generated ($)
- Response rate by persona
- Sequence performance by stage

## Ad-Hoc Reporting
Available on request with 24h turnaround. Ping via Slack or email.
